[id='proc-configuring-jvm-options-{context}']
= Configuring JVM options

.Prerequisites

* {AProductPlatformName} cluster
* A running Cluster Operator

.Procedure

. Edit the `jvmOptions` property in the `Kafka`, `KafkaConnect` or `KafkaConnectS2I` resource.
For example:
+
[source,yaml,subs=attributes+]
----
apiVersion: {KafkaApiVersion}
kind: Kafka
metadata:
  name: my-cluster
spec:
  kafka:
    # ...
    jvmOptions:
      "-Xmx": "8g"
      "-Xms": "8g"
    # ...
  zookeeper:
    # ...
----
+
. Create or update the resource.
+
This can be done using `{cmdcli} apply`:
[source,shell,subs="+quotes,attributes+"]
{cmdcli} apply -f _your-file_

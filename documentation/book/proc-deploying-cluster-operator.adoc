[id='deploying-cluster-operator-{context}']
= Deploying the Cluster Operator to {KubernetesName}

.Prerequisites

ifdef::OpenShift[]
* A user with `cluster-admin` role needs to be used, for example, `system:admin`.
endif::[]
* Modify the installation files according to the namespace the Cluster Operator is going to be installed in.
+
include::frag-cluster-operator-namespace-sed.adoc[]

.Procedure

* Deploy the Cluster Operator:
+
[source,shell,subs="+quotes,attributes+"]
{cmdcli} apply -f install/cluster-operator -n _my-{namespace}_
